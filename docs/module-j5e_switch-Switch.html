new <async> Switch( io [, device ] )

Description
The Switch class allows for control of digital switches
Parameters
Name Type Attributes Default Description
io number | string | object A pin number, pin identifier or a complete IO options object (See Instantiating a Device
device object <optional>
 {} An object containing device options
Properties
Name Type Description
isClosed boolean True if the switch is closed (current is flowing)
isOpen boolean True if the switch is open (current is not flowing)
Examples

Use a switch to control an LED

import Switch from "@j5e/switch";
			import LED from "@j5e/led";
			
			(async function() {
			  const switch = await new Switch(12);
			  const led = await new LED(13);
			
			  switch.on("open", function() {
			    led.off();
			  });
			
			  switch.on("close", function() {
			    led.on();
			  });
			
			})();
